# Animated River – Interactive Canvas Demo

An interactive HTML5 Canvas demo featuring a flowing river and jumping fish.
Users can click on individual fish and assign them custom names using a modern modal interface.

## Live Demo
https://derricktnt.github.io/animated-river/

## Features
- Responsive full-screen Canvas rendering
- Procedurally animated river surface
- Physics-inspired fish jumping motion
- Click interaction with hit detection
- Custom modal UI (no browser prompts)
- Fully client-side (no backend)

## Technology Stack
- HTML5
- CSS3
- JavaScript (ES Modules)
- HTML5 Canvas
- requestAnimationFrame animation loop
